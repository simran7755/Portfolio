Golf Club

  • Designing and implementing web application for Golf club to allow organization to manage tee times, scores and submit new application.

  • The programming tools that used to make this functional website are C#, ASP.Net Core, Razor Pages, HTML, SQL and JavaScript. The software tools are Microsoft Visual Studio and Micrsoft SQL studio.

  • Designing tables, stored procedures with proper data types using Transqt SQL.

  • Handled Sql exception and also programmed validations such as user can't book multiple tee time in a day.

  • Followed SDLC, normalized the given data, and designed different diagrams such as Component view, logical view, interaction and system sequence diagram.
