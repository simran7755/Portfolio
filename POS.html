POS

  • Developed web POS system for hardware shop where items can be added or removed, customer account will be modified and sales will be performed by selling existing items in the system.

  • The programming tools that used to make this functional website are C#, ASP.Net Core, Razor Pages, HTML, SQL and JavaScript. The software tools are Microsoft Visual Studio and Micrsoft SQL studio.

  • Stored procedure and tables were also implemented in database using Transct SQL

  • Server and client side validations are also programmed for this website to make sure user key inputs are valid.
